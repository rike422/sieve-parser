  7. INTRODUCTION

sieve-parser is a pure-ruby implementation for parsing and manipulate the sieve scripts.

  1. REQUIREMENTS

Ruby/SieveParser requires Ruby version 1.9.1 or newer.

  1. INSTALLATION

gem install sieve-parser
